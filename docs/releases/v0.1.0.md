# v0.1.0

Initial V1 baseline release for `llm-wiki`.

## Scope

This release ships the CLI-first Team Memory Wiki foundation described in the spec.

Included commands:

- `init`
- `ingest`
- `query`
- `rebuild-indexes`
- `lint`

## What The Release Supports

- local workspace initialization
- guided ingest for one project/folder at a time
- immutable raw snapshots
- YAML ref manifests
- maintained project cards
- shared markdown indexes
- wiki-first project-orientation query output
- lint-driven review surfaces
- append-only ingest and lint logs

## Operator Model

This release is:

- local
- filesystem-backed
- CLI-only
- wiki-first

This release is not:

- a web service
- a database-backed system
- a frontend application
- a background sync process

## Verification For This Release

Verified on the release commit with:

- `uv run pytest -v`
- `uv run ruff check .`
- `uv run ruff format --check .`
- `uv run mypy src`
- `uv build`
- real CLI smoke scenarios for `init`, `ingest`, `query`, `rebuild-indexes`, and `lint`

## Known Limits

- no explicit reset command for manual overrides
- query does not re-summarize snapshot contents on demand
- duplicate detection is scoped to one guided-ingest slice
- domain is not inferred automatically in V1
- no Docker, DB, API, or frontend surface exists in this release
